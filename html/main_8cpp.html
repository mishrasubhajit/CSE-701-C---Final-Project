int main ( )

key-creates a vector of keys. Some other vectors are also initialized for the later use.

reads the input file as sizeof 8-bit and stores in vector v

computes M^e(mod n) using M.GetPowerMod(e,n) and stores in a vector w

writes the data in the file inout - this is the encrypted file

reads from the encrypted file inout1 and stores in r

computed C^d (Mod n) using C.GetPowerMod(d, n) and stores in a vector<BigInt> object

each element of the vector<BigInt> object is converted into a string using ToString(), then is converted into an ul int using stoul then converted into a char and appended in c

writes the vector<char> object c into output file - the decrypted file
